Description

Designing a statute of a Hip Joint.

Project Scope

  1. Research the anatomy and structure of the human hip joint.
  2. Develop initial sketches and 3D design concepts.
  3. Create a detailed model using CAD software.
  4. 3D print prototypes
  5. Refine the design based on ergonomic and aesthetic considerations.
  6. Prepare drawings for manufacturing.
